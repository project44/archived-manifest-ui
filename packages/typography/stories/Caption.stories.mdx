# Caption

## Usage

<Story name="default" parameters={{ docs: { source: { code: '<Caption>Caption</Caption>' } } }}>
  <Box sx={{ pb: 3 }}>
    <Caption>Caption</Caption>
  </Box>
</Story>

### Variants

The caption comes in two variants:

<Story
  name="variants"
  parameters={{
    docs: {
      source: {
        code: `
<Caption variant="caption1">Caption</Caption>
<Caption variant="caption2">Caption</Caption>
        `,
      },
    },
  }}
>
  <Flex direction="column" gap={2} sx={{ pb: 3 }}>
    <Caption variant="caption1">Caption</Caption>
    <Caption variant="caption2">Caption</Caption>
  </Flex>
</Story>

### Bold

The caption can be rendered bold by using the `isBold` prop:

<Story
  name="bold"
  parameters={{
    docs: {
      source: {
        code: `
<Caption isBold variant="caption1">Caption</Caption>
<Caption isBold variant="caption2">Caption</Caption>
        `,
      },
    },
  }}
>
  <Flex direction="column" gap={2} sx={{ pb: 3 }}>
    <Caption isBold variant="caption1">
      Caption
    </Caption>
    <Caption isBold variant="caption2">
      Caption
    </Caption>
  </Flex>
</Story>
