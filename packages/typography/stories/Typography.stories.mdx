## Installation

```sh
yarn add @manifest-ui/typography
```

## Usage

```tsx
import { Typography } from '@manifest-ui/typography';
```

### Default

By default the typography component renders with a `body2` variant and a size of `medium`.

<Canvas>
  <Story name="Default">
    <Typography>Advanced Supply Chain Visibility</Typography>
  </Story>
</Canvas>
