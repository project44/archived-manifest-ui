## Installation

```sh
yarn add @manifest-ui/layout
```

## Usage

```tsx
import { Box } from '@manifest-ui/layout';
```

### SX prop

Each prop from the `styled-system` are availale in the `sx` prop, additionally standard css can also be specified within the prop.

<Canvas>
  <Story name="Default">
    <Box sx={{ backgroundColor: 'primary.500', height: 200, width: 200 }} />
  </Story>
</Canvas>
