# Grid

Grid is a layout primitive used to easliy create css-grid based layouts.

## Installation

```sh
yarn add @manifest-ui/layout
```

## Usage

```tsx
import { Grid } from '@manifest-ui/layout';
```

### Responsive

The Grid component can easliy support responsive layouts by passing an array value to the column prop.
