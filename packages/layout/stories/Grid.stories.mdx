# Grid

## Usage

<Story
  name="default"
  parameters={{
    docs: {
      source: {
        code: `
<Grid columns={2}>
  <Box />
  <Box />
</Grid>
        `,
      },
    },
  }}
>
  <Grid columns={2}>
    <Box sx={{ backgroundColor: 'primary.500', p: 4 }} />
    <Box sx={{ backgroundColor: 'primary.500', p: 4 }} />
  </Grid>
</Story>

### Width

Specify the width of the columns using the `columnWidth` prop:

<Story
  name="width"
  parameters={{
    docs: {
      source: {
        code: `
<Grid columnWidth="200px">
  <Box />
  <Box />
</Grid>
        `,
      },
    },
  }}
>
  <Grid columnWidth="200px">
    <Box sx={{ backgroundColor: 'primary.500', height: 64 }} />
    <Box sx={{ backgroundColor: 'primary.500', height: 64 }} />
    <Box sx={{ backgroundColor: 'primary.500', height: 64 }} />
    <Box sx={{ backgroundColor: 'primary.500', height: 64 }} />
    <Box sx={{ backgroundColor: 'primary.500', height: 64 }} />
  </Grid>
</Story>

### Responsive

The Grid component can easliy support responsive layouts by passing an array value to the column prop.

  <Story name="responsive">
