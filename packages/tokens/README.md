# @manifest-ui/tokens

**@manifest-ui/tokens** contains values for color, spacing, and typography tokens for use with Manifest UI design system.

<p><br /></p>

```sh
# with npm
npm install @manifest-ui/tokens

# with yarn
yarn add @manifest-ui/tokens
```
