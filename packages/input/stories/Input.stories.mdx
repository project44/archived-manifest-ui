### Sizes

The input comes with two sizes:

<Story
  name="sizes"
  parameters={{
    docs: {
      source: {
        code: `
<Input placeholder="Small" size="small" />
<Input placeholder="Medium" size="medium" />
        `,
      },
    },
  }}
>
  <Flex direction="column" gap={2} sx={{ pb: 3 }}>
    <Input placeholder="Small" size="small" />
    <Input placeholder="Medium" size="medium" />
  </Flex>
</Story>

### Icons

The input can be decorated with icons using the `startIcon` and `endIcon` props:

<Story
  name="icons"
  parameters={{
    docs: {
      source: {
        code: `
<Input placeholder="Default" startIcon={<Person />} />
<Input endIcon={<Search />} placeholder="Default" />
<Input endIcon={<Search />} placeholder="Default" startIcon={<Person />} />
        `,
      },
    },
  }}
>
  <Flex direction="column" gap={2} sx={{ pb: 3 }}>
    <Input placeholder="Default" startIcon={<Person />} />
    <Input endIcon={<Search />} placeholder="Default" />
    <Input endIcon={<Search />} placeholder="Default" startIcon={<Person />} />
  </Flex>
</Story>

### Disabled

The input can be disabled using the isDisabled prop:

<Story
  name="disabled"
  parameters={{ docs: { source: { code: '<Input isDisabled placeholder="Disabled" />' } } }}
>
  <Box sx={{ pb: 3 }}>
    <Input isDisabled placeholder="Disabled" />
  </Box>
</Story>
