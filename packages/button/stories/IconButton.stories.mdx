# IconButton

## Usage

<Story
  name="default"
  parameters={{ docs: { source: { code: '<IconButton><Add /></IconButton>' } } }}

### Variants

The icon button comes with three variants:

<Story
  name="variants"
  parameters={{
    docs: {
      source: {
        code: `
<IconButton variant="filled"><Add /></IconButton>
<IconButton colorTheme="neutral" variant="outlined"><Add /></IconButton>
<IconButton colorTheme="neutral" variant="text"><Add /></IconButton>
        `,
      },
    },
  }}
>
  <Flex align="center" gap={2} sx={{ pb: 3 }}>
    <IconButton variant="filled">
      <Add />
    </IconButton>
    <IconButton colorTheme="neutral" variant="outlined">
      <Add />
    </IconButton>
    <IconButton colorTheme="neutral" variant="text">
      <Add />
    </IconButton>
  </Flex>
</Story>

### Danger

A button can be styled to signify destructive actions using the `colorTheme` prop:

<Story
  name="danger"
  parameters={{
    docs: { source: { code: '<IconButton colorTheme="danger"><Add /></IconButton>' } },
  }}
>
  <Box sx={{ pb: 3 }}>
    <IconButton colorTheme="danger">
      <Add />
    </IconButton>
  </Box>
</Story>

### Disabled

The icon button can be disabled using the `isDisabled` prop:

  parameters={{
